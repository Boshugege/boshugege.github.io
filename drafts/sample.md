这是一个示例段落，包含行内代码 `x = 42`，以及行内数学 $e^{i\theta}=\cos\theta + i\sin\theta$。

下面是一个 Python 代码块：

```python
def fib(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

下面展示一个独立数学公式：

$$
\int_0^{\infty} e^{-x^2}\,dx = \frac{\sqrt{\pi}}{2}
$$
