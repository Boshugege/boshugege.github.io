起步

亲爱的 Codex 先生在帮我生成页面时写的示例文本是这样一句话，我自勉：

想把写作这件事简化成“每天留一点痕迹”。
今天先把页面和流程搭起来，哪怕只写两三句也算完成。

然而本来想在图书馆写点什么再走的，突然发现马上就要上课了，whatever，又写不动了。

回到宿舍，还是打算拾起这个活动，毕竟每天写东西就和戒烟一样，是次数多而时长短的。而且产出的内容最后多多少少会变成流水账，累晕了的时候没空动脑思考

前段时间用 Copilot 和 Codex 重写了这个博客。

这个博客挂在 GitHub Pages 上，一直是一个静态的站点。小时候刚开始玩用 Hexo 拉了个模版发现搞不明白，彼时又没有 AI 这类强大的工具，于是怒而自己从 HTML 和 CSS 写起。然而我实在不是一个有毅力的开发者，也不是一个有毅力的写作者，所以在网课期间用上课摸鱼的时间折腾的好看了一点，填上了一些随便写的内容和随便写的作业，盒模型没怎么搞明白，比较有成就感的是放了一个 Spotify 的小窗。期间似乎还被一位同学 fork 了，不过我也没搞清楚那位同学是谁。

上了高中感觉自己折腾的实在不是特别好看，而且直接与富文本搏斗实在不是优雅和现代的方法。于是用 Gatsby 和一个它的模板继续开发，但是其实只是套了模板随意改改，仍然不是很明白。想来在当时的那种好奇心下，要是我能有足够的时间去折腾，多少也能有点理解，可惜现在被 AI 接管了大脑，没有那种钻研的动力了。

前段时间的重写是看了别人（Master Hash, 暂译为哈希主人，除非本人看到此文要求我修改）的博客，多少也想重新捡起自己手上的东西。然而我实在对 coding 毫无理解和想法，所以只能采取 文科生进军 GitHub 的做法，让 AI 大人接管。而极简白板的风格显然是要伪装成上古大佬。（这种用 python 脚本生成的玩意儿，这种 site.js 打包成一大坨的做法，应该上古大佬都不屑于骂人。）把一些原本的文章迁移过来，太黑历史的就丢掉了，当然它们还是会安安静静地躺在我的 git 历史里的。

因为发现自己没能力一直写文章，所以就侧重于这个随想。但是我肯定知道我写不了多少，希望能多努力努力吧。
